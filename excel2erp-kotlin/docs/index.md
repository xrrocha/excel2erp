# De la Aritmética al Álgebra

[![TL;DR](img/tldr.png)](img/tldr.png)

Un script rápido para convertir pedidos Excel al formato del ERP funcionó... hasta que cada nuevo cliente trajo su propio formato. La solución "copiar y pegar" colapsó bajo su propio éxito.

- El código **"aritmético"** (copiar-pegar) crea deuda técnica inevitable
- La alternativa **"algebraica"** usa metadatos declarativos (YAML) para describir *qué* extraer, no *cómo*
- Resultado: soportar un nuevo formato Excel requiere solo editar configuración
- Herramientas: **Kotlin** + **HTMX** + **YAML**
